实验五  队列的基本操作

一、实验目的

1、熟练掌握顺序队、链队的存储原理；

2、熟练掌握顺序队、链队的Java语言实现方法；

3、掌握队列空、队满的判断条件；

4、了解循环队列与普通队列实现上的不同及其解决方法。

[问题描述]
